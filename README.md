##Como usar o jogo (código em p5.js)

Abrir no editor p5.js
Copie e cole o código completo no editor.
Executar o jogo

Clique em Play ▶️ no editor.
A tela inicial aparece com instruções.
##Controles:
Use as setas para cima e para baixo para mover o quadrado azul.
Desvie dos retângulos vermelhos que vêm da direita.
Pontuação e HUD
O HUD no topo mostra Score, Dificuldade e Tempo.

##Game Over
Quando o jogador colide com um obstáculo, o quadrado azul se despedaça em fragmentos.
Apenas o obstáculo atingido brilha em amarelo.
A tela mostra a pontuação e o tempo jogado.

##Pressione ENTER para reiniciar.
